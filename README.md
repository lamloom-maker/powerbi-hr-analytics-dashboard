# HR Analytics Dashboard (Power BI)

This project presents an interactive HR Analytics dashboard built using Power BI.

## Objective
To analyze employee data and provide insights into:
- Workforce distribution
- Attrition rate
- Employee satisfaction
- Salary and performance metrics

## Key Metrics
- Total Employees
- Attrition Rate
- Average Satisfaction Score
- Average Salary
- Average Performance Score

## Visualizations
- Employee count by department
- Gender distribution
- Nationality distribution
- Interactive slicers (Department, Gender, Employment Type)

## Tools
- Power BI
- Excel (Data Source)

## Notes
This project is intended for professional portfolio demonstration purposes.
